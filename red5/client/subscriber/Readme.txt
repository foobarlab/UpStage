Compile:

Compile using FlashDevelop IDE.
Set environment variable FLASHPLAYER_DEBUGGER to the appropriate location (e.g. "C:\develop\bin\flashplayer_11_sa_debug.exe").

You find the compiled binary swf here: "/server/html/media/stream.swf".
For testing you can also check "/html/subscriber.html" after compilation.


Running Red5:

Tested using Red5-1.0.1. You need to install "oflaDemo".
